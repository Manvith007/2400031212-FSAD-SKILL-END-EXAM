## ⚙️ How to Run the Project

### 1. Clone the Repository

```
git clone https://github.com/Manvith007/2400031212-FSAD-SKILL-END-EXAM.git
```

### 2. Navigate to Project Folder

```
cd 2400031212-FSAD-SKILL-END-EXAM
```

### 3. Install Dependencies (if Node.js)

```
npm install
```

### 4. Run Backend Server

```
npm start
```

### 5. Open Frontend

Open `index.html` in browser
OR run using Live Server
